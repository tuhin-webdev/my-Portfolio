# MD SAHAB UDDIN (TUHIN) Portfolio

A modern Full Stack Developer portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 Live Demo
https://tuhindev.netlify.app/

## 🛠️ Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- TypeScript

## ✨ Features
- Responsive Design
- Modern UI/UX
- Smooth Animations
- Project Showcase
- Contact Section

## 📦 Installation

```bash
git clone <(https://github.com/tuhin-webdev/my-Portfolio)>
cd my-Portfolio
npm install
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
